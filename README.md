# Saanvi Raghavendran — Portfolio

A single-page React portfolio showcasing AI, music technology, and social impact work by a Grade 11 scholar who loves to build cultural tools and experiences.

## Tech stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Development

1. `npm install`
2. `npm run dev` (run the Vite dev server and open the URL shown in the terminal)
3. `npm run build` (produce the optimized production bundle in `dist/`)

The `public/placeholder.svg` asset is used as the favicon and can be swapped out if needed. Static files from `public/` are copied as-is into the production build.

## Resources

- The source entrypoint for the React app lives under `src/main.tsx`.
- Tailwind configuration is in `tailwind.config.ts`.
- Linting and formatting rules can be found in `eslint.config.js` and `postcss.config.js`.
